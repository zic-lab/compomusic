Avertissement important sur "Smart screen"

  • Si la protection « Smart Screen » est activée sur votre PC, il est possible que Windows ne vous encourage pas à installer le logiciel. « Smart Screen » est un système de « Protection fondée sur la réputation » : Windows considère qu'une absence de réputation peut constituer une absence de bonne réputation, ce qui, dans le cas des présents logiciels (et de beaucoup d'autres), n'est pas exact.
  • Toutefois, pour Windows, cet avertissement ne constitue pas une interdiction d'installation définitive. Le formulaire « Smart Screen » autorise la poursuite de l'installation.

Actions recommandées pour l'installation

  • Avant installation et pour dissiper toute suspicion, vous pouvez  soumettre à votre anti-virus le fichier msi que vous venez de télécharger. Faites un clic droit sur le fichier et lancez l'antivirus.
  • Après la vérification du logiciel par l'anti-virus, vous êtes prêt à installer le logiciel : pour ce faire, cliquer sur le fichier msi téléchargé.
  • La fenêtre « Smart Screen » peut apparaître alors : cliquez sur le lien Informations complémentaires afin d' Exécuter quand même l'installation du logiciel.
  • Poursuivre l'installation de l'application en suivant le guide d'installation du fichier msi

Autre démarche

  • Il est aussi possible de "tester" préalablement un logiciel dans le bac à sable de windows (sandbox). Pour ce faire télécharger un logiciel directement à partir du bac à sable à l'aide du navigateur de ce bac à sable.
  • La même démarche est possible avec une machine virtuelle spécifique installée sur votre PC (VirtualBox ou autre).

Solution la plus simple

Nous tenons à faire remarquer que l'utilisation de Smart Screen n'a pas un caractère obligatoire pour Microsoft :
  • En effet, nous avons déjà vu qu'il est possible d'outrepasser le Smart Screen en cliquant sur le lien Exécuter quand même lors de l'installation.
  • De plus, Microsoft propose de manière très officielle, dans la configuration de Windows de désactiver le Smartcreen.
Donc, Microsoft permet à 2 reprises la neutralisation du Smart Screen. Nous en concluons que la désactivation du Smart Screen dans le système Windows peut être raisonnablement envisagée. Ceci est la solution que nous nous permettons de recommander en premier lieu.

Désactivation du Smart Screen

Pour désactiver le filtre SmartScreen dans Windows, suivez ces étapes :

  1. Ouvrez les Paramètres de Windows : Appuyez sur les touches Windows + I pour ouvrir les Paramètres.
  2. Aller dans la recherche : Recherchez le terme "Protection fondée sur la réputation"
  3. Désactiver "Vérifier les applications et les fichiers"
