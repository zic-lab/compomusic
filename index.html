Le but de cette page est de fournir un téléchargement de ces logiciels.

Avertissement important sur "Smart screen"

Si la protection « Smart Screen » est activée sur votre PC, il est possible que Windows ne vous encourage pas à installer le logiciel. « Smart Screen » est un système de « Protection fondée sur la réputation » : Windows considère qu'une absence de réputation peut constituer une absence de bonne réputation, ce qui, dans le cas des présents logiciels (et de beaucoup d'autres), n'est pas excacte.
Toutefois, pour Windows, cet avertissement ne constitue pas une interdiction d'installation définitive. Le formulaire « Smart Screen » autorise la poursuite de l'installation.

Actions recommandées pour l'installation

  1. Avant installation et pour dissiper toute suspicion, vous pouvez  soumettre à votre anti-virus le fichier msi que vous venez de télécharger. Faites un clic droit sur le fichier et lancez l'antivirus.
  2. Après la vérification du logiciel par l'anti-virus, vous êtes prêt à installer le logiciel : pour ce faire, cliquer sur le fichier msi téléchargé.
  3. La fenêtre « Smart Screen » peut apparaître alors : cliquez sur le lien « Informations complémentaires » afin d'« Exécuter quand même » l'installation du logiciel.
  4. Poursuivre l'installation de l'application en suivant le guide d'installation du fichier msi

Autre démarche

  1. Il est aussi possible de "tester" prélablement un logiciel dans le bac à sable de windows (sandbox). Pour ce faire télécharger un logiciel directement à partir du bac à sable à l'aide du navigateur de ce bac à sable.
  2. La même démarche est possible avec une machine virtuelle spécifique installée sur votre PC (VirtualBox ou autre).
